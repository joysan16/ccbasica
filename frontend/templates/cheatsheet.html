🧠 Dynamic Memory
malloc · calloc · realloc · free — one page
🔨The 4 Functions
FunctionWhat it doesInitialized?
malloc(size)Allocates size bytes❌ Garbage values
calloc(n,size)Allocates n×size bytes✅ Zero-initialized
realloc(ptr,new)Resizes existing blockNew part uninitialized
free(ptr)Releases memory—
🐛Memory Leak
→Allocated but never freed
→Memory usage grows forever
⚠️ Fatal in long-running embedded systems (routers, IoT)
💥Double Free / Dangling
→free(p) twice = undefined behavior
free(p); p = NULL;
→Always NULL after freeing!
✅Safe realloc Pattern
int *temp = realloc(p, newSize);
if(temp != NULL) p = temp;
⚠️ Never write p = realloc(p, ...) directly — if it fails, you lose the original pointer (memory leak)!
